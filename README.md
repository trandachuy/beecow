## GosellDashboard

Default port: 8080

`Node` version required: >= 12

## Available Scripts

In the project directory, you can run:

### Development mode

`yarn start-dev`

`yarn start-qa`

`yarn start-staging`

`yarn start-prod`

### Production mode

`yarn build-dev`

`yarn build-qa`

`yarn build-staging`

`yarn build-prod`

Note:
- Only build-dev include source-map (eval).
- Only build-prod include minimize images plugin.
